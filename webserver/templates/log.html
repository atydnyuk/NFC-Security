Webserver contact log

Every time that a device contacts the webserver and says that it has a code for it that it got from an NFC tag, that message gets entered into the log. These log messages signify attempts to submit a scan. In order to differentiate them, the messages that are confirmed to be valid will be hilighted in green. Messages that are not accepted as scans will be grey, and rejected requests will be red.
