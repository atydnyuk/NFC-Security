NFC Webserver

Every time that a device contacts the webserver at address "/submit" we assume that it is a device that has scanned an NFC tag. The device submits the code that it was given by the tag in the request, and this is recorded in our log. If the code is valid, then the device gets a code sent to it that will allow that device to confirm its scan of the tag. The webserver will in turn store that information so that the same device will not get multiple codes.
